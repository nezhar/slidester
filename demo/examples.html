Basic Slider

This is the most basic example of how you can use Slidester.

Configuration:
  • JavaScript
  • HTML
$(function() {
    $("#example1").slidester();
}
<div id="example1" class="slider">
    <div class="images">
        <img src="slides/1.jpg">
        <img src="slides/2.jpg">
        <img src="slides/3.jpg">
        <img src="slides/4.jpg">
    </div>
</div>

Customized Slider

You can also apply customized configrations to the slider so it gets closer to your requrenments. You have to do this is the ...

Configuration:
  • JavaScript
  • HTML
$("#example2").slidester({
    direction: -1,
    startSlide: 3,
    animation: "fade",
    initHeightSpeed: 1500,
    initSlideSpeed: 1500,
    heightSpeed: 1000,
    slideSpeed: 4000,
    animationSpeed: 2000,
    pauseOnHover: true,
    controlButtons: false,
    controlRadio: true,
    controlThumbnails: false,
    captions: true,
    captionsDisplayClass: null
});
